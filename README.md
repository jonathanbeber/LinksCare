# LinksCare
Ferramenta para monitoramento de links

# Uso
Usage:

    linkscare [OPCOES] URL

    OPCOES:

        -n            Não fica monitorando, apenas verifica uma vez

    ARGUMENTOS:

        URL           URL para monitorar
